# Autonomous Navigation and Grasping System for Unmanned Robots
## Description
This is used for the final project of Robotic Perception and Intelligence. The software system is based on the Linux/Ubuntu system and the ROS2 communication framework.

These codes are about the part that I am responsible for, which are all about robot arm control.

The overall code of the project is on the robot's computer and it has temporarily gone missing. But the framework of the entire project is based on open source:
